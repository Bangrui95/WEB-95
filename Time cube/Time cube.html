TouchDesigner & Google MediaPipe

This project integrates Google MediaPipe with TouchDesigner to create a computer-vision-based interactive visual system. Using a camera as the primary input device, the system continuously captures human movement and translates it into real-time visual responses, creating an intuitive and immersive interaction between the participant and the digital environment.

During the experimental phase, the project focuses on MediaPipe’s Hand Tracking model to detect and track hand movements. The system identifies key finger joint landmarks in real time and transfers this spatial data into TouchDesigner for further processing and mapping. By analysing the positions of finger joints, gesture shapes, and movement trajectories, hand gestures can directly control the generation, transformation, and motion of visual elements.

Applications & Collaboration

This experiment explores the integration of computer vision technology and real-time visual art, demonstrating the potential of gesture recognition as a control interface for interactive visual systems. In future applications, the system could be expanded into dynamic posters, interactive installations, immersive visual environments, and live audiovisual performances.
